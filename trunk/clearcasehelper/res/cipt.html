Code Inspection Package Tool (CIPT)
 

CIPT has been upgraded to version . Please refer to the Release Notes for details of changes included in this release.
What is CIPT?
CIPT is a web based Code Inspection Package Tool that provides delta code line count  based on a Sablime MR, a ClearQuest CR or combination of branch type name and vob location. The main purpose of this tool is for code inspection preparation.
CIPT FAQ

It is recommended that you read the CIPT Frequently Asked Questions (FAQ) first, before opening an SR against the tool.
